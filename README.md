# EQ-WISDOM (Children Edition)
> 播下情商的种子，培养更聪明、更具共情力的孩子。

## 项目简介
本项目致力于为儿童提供系统的情商（EQ）培养方案。涵盖情绪认知、自我调节、社交技巧及共情能力培养。

## 核心内容
- **情绪地图**: 帮助孩子识别 20 种基础情绪。
- **互动练习**: 日常生活中的情商实践小游戏。
- **绘本推荐**: 针对不同年龄段的精选书单。

## 如何参与
欢迎教育者、家长及心理学家提交 Issue 或 Pull Request 来完善内容。
